DDD + CQRS + Event Sourcing + API Platform
==========================================

Demo application wiring up some concepts and libraries :
- DDD
- CQRS
- Event Sourcing
- API Platform
- Prooph

Install
=======

    $ git clone https://github.com/Lctrs/apiplatform-ddd-cqrs-es-demo.git
    $ docker-compose up -d

And go to https://localhost.
